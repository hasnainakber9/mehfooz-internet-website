### Installation

1. Clone the repository:
```bash
git clone https://github.com/hasnainakber9/mehfooz-internet-website.git
cd mehfooz-internet-website
```

2. Open `index.html` in your browser:
```bash
# On macOS
open index.html

# On Linux
xdg-open index.html

# On Windows
start index.html
```

Or use a local server:
```bash
# Python 3
python -m http.server 8000

# Node.js (if you have http-server installed)
npx http-server
```
